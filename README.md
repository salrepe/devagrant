devagrant
=========

A vagrant box with a minimum setup

## Requirements

* [VirtualBox](https://www.virtualbox.org)
* [Vagrant](http://vagrantup.com)

## Starting the machine

```zsh
$ git clone https://github.com/salrepe/devagrant.git
$ cd devagrant
$ vagrant up
$ vagrant provision
$ vagrant ssh
```
